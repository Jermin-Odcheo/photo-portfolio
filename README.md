# photo-portfolio

A photography portfolio built with **Next.js**, **TypeScript**, and **Tailwind CSS** — showcasing landscapes, portraits, street photography, nature, and architecture.

## Features

- 🖼 **Gallery** — Masonry grid with category filtering
- 🏠 **Home** — Full-screen hero, featured shots, and stats
- 👤 **About** — Bio, gear list, and tech stack
- ✉️ **Contact** — Enquiry form with success state
- 📱 **Responsive** — Mobile-first design
- 🌑 **Dark theme** — Minimal, editorial aesthetic

## Tech Stack

- [Next.js 16](https://nextjs.org/) — App Router
- [TypeScript 5](https://www.typescriptlang.org/)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [React 19](https://react.dev/)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run start` | Start production server |
| `npm run lint` | Run ESLint |
